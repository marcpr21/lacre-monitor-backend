The Lacre Monitor app is a mobile photo monitoring tool for employees. Key features include individual user login, in-app camera-only photo capture with strict time-based rules for Seal Photos (M/W/F, 06:00-12:00) and Meter Photos (Daily, 06:00-09:00, 17:00-18:00). Photos are temporary and not saved on the phone, and push notifications remind employees of missed submissions. The admin panel allows viewing all photos hierarchically (Date > Employee > Type).

- Improved Admin Panel with user management (add, edit, delete, reset passwords), a Relatório de Conformidade (compliance report showing missed submissions per employee, day, and photo type).

- A new system for administrators to authorize employees to take photos outside scheduled periods (individual per employee, per photo type: lacre, medidor manhã/tarde).
- A system for configurable email alerts to admins when specific employee/photo type combinations are submitted.
- The backend has been moved to Railway for 24/7 availability and is currently being migrated to MongoDB Atlas for persistent data storage to prevent data loss on server restarts.
